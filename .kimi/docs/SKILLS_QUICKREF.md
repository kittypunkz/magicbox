## 🎯 Each Agent's Top 3 Skills

### PM (Product Manager)
1. **Requirements Analysis** - Understand what to build
2. **Communication** - Coordinate team + user
3. **Project Management** - Keep on track

### Architect
1. **System Design** - Design scalable solutions
2. **Technology Expertise** - Know the stack deeply
3. **Security Architecture** - Design securely

### Developer
1. **Programming Excellence** - Write great code
2. **Frontend/Backend Dev** - Full-stack capability
3. **Error Handling** - Build resilient apps

### QA
1. **Testing Strategy** - Know what to test
2. **Code Review** - Spot issues
3. **Security Testing** - Find vulnerabilities

- 🧪 **QA**: Depth in quality + breadth in testing types
